# homebrew-tap

Homebrew tap for [@oswaldoacauan](https://github.com/oswaldoacauan) tools.

## Usage

```bash
brew install oswaldoacauan/tap/<formula>
```

## Formulas

- **diffgotchi** — Opinionated TUI diff reviewer ([repo](https://github.com/oswaldoacauan/diffgotchi), [site](https://diffgotchi.dev))
  ```bash
  brew install oswaldoacauan/tap/diffgotchi
  ```
